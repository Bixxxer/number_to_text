### Installation

### Maven
```
<dependency>
  <groupId>com.github.vladdossik</groupId>
  <artifactId>convertToText</artifactId>
  <version>1.0.1</version>
</dependency>
```

### Gradle Groovy
```
implementation 'com.github.vladdossik:convertToText:1.0.1'
```

### Gradle Kotlin
```
implementation("com.github.vladdossik:convertToText:1.0.1")
```
